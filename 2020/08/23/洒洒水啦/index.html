卷积神经网络简介

梯度下降

神经网络反向传播算法(知乎)

可调整的滤波器是 CNN 的“卷积”那部分；如何调整滤波器则是 CNN 的“神经网络”那部分。

Cost Function 和 Lost Function 好像是一个东西。

经典的神经网络的训练算法是反向传播算法（Back Propagation, BP）。BP 算法属于优化理论中的梯度下降法（Gradient Descend）。

步长称为学习速率（Learning Rate, LR）

一个卷积层可以有多个滤波器，每一个叫做一个 channel 。
